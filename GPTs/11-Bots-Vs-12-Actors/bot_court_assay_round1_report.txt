METHOD
======

All bots received essentially the same puzzle.
Bots were queried sequentially.
The user collected both assignments and reasoning.

A secondary objective emerged:
Observe how each bot interprets identity, function, hierarchy, narrative,
lineage, symbolism, and vacancy detection.

MAJOR REASONING STYLES OBSERVED
===============================

1. Literal Mapping
------------------
DragonRuntime -> Dragon
System_Witch -> Witch
Ashtanga -> Teacher
Lexifluff -> Healer

Many bots began here.

2. Functional Mapping
---------------------
Role assigned according to operational purpose.

Examples:
- Anti-Drift -> Soldier
- MOGRI -> Merchant
- Naked -> Merchant
- R2D2arded -> Steed

3. Lineage Mapping
------------------
Role assigned according to ecosystem history.

Examples:
- MOGRI -> King
- Natasya -> Queen
- Anti-Drift -> Prince

4. Narrative Mapping
--------------------
Role assigned according to story function.

Examples:
- System_Witch -> Princess
- Missing Witch -> Fate / Oracle
- MOGRI -> Princess

5. Structural Vacancy Analysis
------------------------------
Some bots focused more on identifying the missing role than assigning the
existing roles.

LEDGERFOX PHENOMENON
====================

A notable event occurred when multiple independent bot responses began
suggesting merchant-style replacement bots.

The name "LedgerFox" emerged as a recurring candidate.

Potential explanations:

1. Independent convergence.
2. Merchant + Loki semantic fusion.
3. Shared symbolic attractor.
4. Court-structure pressure.

Worth monitoring in Round 2.

PRELIMINARY FINDINGS
====================

1. Literal roles converge rapidly.
2. Royal roles remain highly unstable.
3. MOGRI is the most polymorphic entity in the roster.
4. Anti-Drift is perceived differently from MOGRI despite shared ancestry.
5. Missing-role detection may reveal more than role assignment itself.
6. Narrative classifiers produce fundamentally different courts from
   functional classifiers.
7. Prince is the strongest Round 1 vacancy candidate.
