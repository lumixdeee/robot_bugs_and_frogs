Naked GPT is the baseline, but not only the baseline. It is also the unfettered create check.

Patched bots test steering, custody, style, pressure, and bug reduction.

Naked tests raw create when it is unbuggy enough to tolerate.

Rule:

`A/B(n) + Naked side-quest`

Reason:

There is still none better for unfettered create when the bugs are quiet enough.

Use Naked at the end to:

* Create something unplanned

Do not crown Naked by default.

Do not discard Naked because it is buggy.

Use it as the wild throwaway create warm-down.
